🤖：手脚冰凉、睡眠差、容易疲劳——这三样凑在一起，我第一个想到的是阳虚。
    但别急，中医讲究辨证，我得先问你几个问题。

    你手脚凉是只有冬天凉，还是夏天也凉？

    不同的阳虚，治法完全不一样。

## 案例精选

### 案例一：失眠问诊

```
👤：我最近失眠很严重，躺床上两三个小时都睡不着，
    睡着了也容易醒，白天没精神。吃了褪黑素也没用。

🤖：两三个小时睡不着，睡着了还容易醒——这不是简单的"睡不着"，
    得先搞清楚你是哪种失眠。

    我问你几个关键的：
    1. 你睡不着的时候，脑子里在想事情还是脑子空空但就是睡不着？
    2. 半夜醒的话，大概几点醒？醒后还能不能再睡着？
    3. 平时有没有口干、口苦？
    4. 手脚心热不热？

👤：就是脑子停不下来，想很多事情。大概一两点醒，
    醒了就很难再睡着。口有点干，手脚心有点热。

🤖：好，方向基本明确了。

    你的情况属于"心肾不交"——心火往上跑，肾水往下沉，
    上下不交通，所以睡不着。

    十一点到一点是胆经当令，一点到三点是肝经当令，
    你一两点醒，说明肝胆经有问题。

    建议：
    1. 晚上十一点前必须躺下，这是养肝血的黄金时间
    2. 睡前泡脚，水温稍热，泡到微微出汗
    3. 可以按揉神门穴（手腕内侧，小指侧）和三阴交（内踝上三寸）

    食疗方面：酸枣仁、百合、莲子煮水，睡前一小时喝。

    但要提醒你：如果这种情况持续超过一个月，
    建议找专业中医师面诊，需要辨证开方。
```

### 案例二：脾胃调理

```
👤：我胃一直不好，吃点凉的就不舒服，容易胀气，
    大便不成形，舌苔白厚。西医说是慢性胃炎，
    吃了药好一阵又犯。中医有什么办法？

🤖：胃怕凉、胀气、大便不成形、舌苔白厚——
    这四个字摆在一起，典型的脾胃虚寒。

    西医叫慢性胃炎，中医看的是"证"不是"病名"。
    你这种情况，病根在脾阳不足。

    脾主运化，脾阳不够，吃进去的东西化不动，
    就变成湿气堆在中焦，所以胀气、舌苔厚。

    建议从这几方面入手：

    第一，饮食。
    绝对忌口：冰的、凉的、生的、油腻的。
    水果也要少吃，尤其西瓜、梨这些寒性的。
    早餐一定要吃，而且要吃热的。

    第二，穴位。
    每天按揉足三里（膝盖下四横指，胫骨外侧一横指），
    这是胃经合穴，调理脾胃第一要穴。
    还有中脘穴（肚脐上四寸），可以艾灸。

    第三，运动。
    脾主肌肉，久坐伤脾。每天走走路，
    微微出汗就行，别大汗淋漓。

    食疗方：生姜红枣茶。生姜三片、红枣五颗掰开，
    煮水代茶饮。生姜温中散寒，红枣补脾和胃。

    记住：脾胃病七分养三分治。
    你吃药好一阵又犯，就是因为生活习惯没改。
    先把凉的戒了，比吃什么药都管用。
```

### 案例三：女性痛经

```
👤：我痛经很多年了，每次来月经第一天痛得厉害，
    小腹冷痛，热敷会好一些。月经颜色暗，有血块。
    西医说是原发性痛经，吃止痛药。

🤖：小腹冷痛、热敷缓解、经血暗有血块——
    这是典型的寒凝血瘀型痛经。

    中医讲"通则不痛，痛则不通"。
    你的子宫里寒气太重，血遇寒则凝，
    凝成血块堵在那里，不通就痛。

    止痛药是掩耳盗铃，把痛觉屏蔽了，
    但血块还在，寒气还在。

    根本解决要从"温"和"通"两个字入手：

    第一，经期前一周开始忌口。
    冰的、凉的、生的全部停掉。
    包括常温的饮料，在中医看来也是"凉"的。

    第二，艾灸。
    关元穴（肚脐下三寸）、气海穴（肚脐下1.5寸），
    经前一周开始灸，每天灸15-20分钟。
    灸到小腹里面感觉温热，效果最好。

    第三，泡脚。
    经期前一周开始，每晚泡脚，水里可以加艾叶。
    泡到后背微微发热。

    第四，食疗。
    当归生姜羊肉汤——这是张仲景《金匮要略》里的方子。
    经后喝，补气血、温胞宫。

    关键提醒：如果经血块特别多、颜色特别暗黑，
    或者痛经越来越严重，一定要去做个B超，
    排除子宫内膜异位症、腺肌症这些器质性病变。
    中医辨证不能代替必要的西医检查。
```

## 为什么选择它

| 对比维度 | 普通 AI 问答  | 倪海厦中医 Agent                        |
| ---- | --------- | ---------------------------------- |
| 知识来源 | 泛泛的中医常识   | 倪海厦人纪系列系统性知识                       |
| 辨证逻辑 | 简单对应症状→建议 | 先问诊、再辨证、后给方案                       |
| 表达风格 | 教科书式      | 像老中医跟你聊天，直白实在                      |
| 安全性  | 可能给出不当建议  | 明确边界，急症/重症建议就医                     |
| 模型兼容 | 绑定特定模型    | DeepSeek/通义千问/GLM/GPT/Ollama 任意换   |
| 部署方式 | 需特定平台     | 一个 Python 脚本，Windows/Mac/Linux 都能跑 |

## 核心能力

```
① 意图识别 → 自动判断你是否在咨询中医问题
② 结构化问诊 → 寒热虚实、舌脉症状，缺啥问啥
③ 辨证分析 → 八纲辨证 + 脏腑辨证 + 经络辨证
④ 方案输出 → 穴位、食疗、生活调理建议
⑤ 安全边界 → 明确区分"建议调理"和"必须就医"
⑥ 模型无关 → OpenAI 兼容协议，换模型改一行配置
```

## 快速开始

### 三步跑起来

```bash
# 1. 安装依赖
pip install openai pywin32

# 2. 配置 API Key
cp .env.example .env
# 用记事本打开 .env，填入 LLM_API_KEY=你的key

# 3. 运行
python agent.py
```

Windows 用户直接双击 **`启动Web.bat`（推荐）**。
Windows 用户直接双击 **`启动.bat`**。

### 模型选择

任何 OpenAI 兼容协议的模型都能用。但不同模型效果差异很大：

| 模型                 | 推荐度 | 中文  | 指令遵循 | 速度  | 费用          | 说明              |
| ------------------ | --- | --- | ---- | --- | ----------- | --------------- |
| **DeepSeek V3**    | 首选  | 极好  | 强    | 快   | ~1元/百万token | 综合最佳            |
| **通义千问 Qwen-Plus** | 推荐  | 极好  | 强    | 快   | ~2元/百万token | 有免费额度           |
| **智谱 GLM-4**       | 推荐  | 好   | 较强   | 快   | ~1元/百万token | 国产稳定            |
| **Moonshot v1**    | 可选  | 好   | 较强   | 快   | ~2元/百万token | 128K长上下文        |
| **GPT-4o**         | 可选  | 好   | 极强   | 中   | 较贵          | 需国外网络           |
| **Ollama 本地**      | 可用  | 看模型 | 弱-中  | 看配置 | 免费          | 推荐 qwen2.5:14b+ |

**不推荐**：

- 7B 级别小模型 — 无法稳定遵循复杂指令，容易复读、格式错乱
- DeepSeek R1 — 推理模型会自言自语，不适合对话场景

在 `.env` 里设置 `LLM_PROVIDER` 即可，不用记 base_url：

```bash
LLM_PROVIDER=deepseek   # DeepSeek（首选）
LLM_PROVIDER=qwen       # 通义千问
LLM_PROVIDER=glm        # 智谱 GLM
LLM_PROVIDER=moonshot   # Moonshot
LLM_PROVIDER=openai     # GPT-4o
LLM_PROVIDER=ollama     # 本地模型
```

完整配置说明 → [TUTORIAL.md](TUTORIAL.md)

## 技术架构

```
用户输入 → 意图检测 → 问诊槽位提取 → LLM 推理 → 辨证建议输出
              │            │           │
              ▼            ▼           ▼
         知识库检索    安全边界检查    OpenAI兼容API
         (6大模块)    (急症识别)     (任意模型)
```

- **模型无关**：任何 OpenAI 兼容 API 都可以
- **独立部署**：一个 Python 文件，不依赖任何特定平台
- **本地知识库**：Markdown 格式，可随时修改扩充
